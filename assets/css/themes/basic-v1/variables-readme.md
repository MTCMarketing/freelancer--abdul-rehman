| **Variable**                                    | **Description**                                                                 |
|-------------------------------------------------|---------------------------------------------------------------------------------|
| `--color-background`                            | Base background color for the site.                                             |
| `--color-text`                                  | Default text color.                                                             |
| `--color-link`                                  | Color for links.                                                                |
